# PDF Analyzer Bot

A Streamlit app that lets users upload a PDF, generate quiz prompts, and ask questions using sentence-transformer embeddings.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Community Cloud

Use these values when creating the app:

- Repository: your GitHub repository for this project
- Branch: `main`
- Main file path: `app.py`

The app dependencies are declared in `requirements.txt`.
